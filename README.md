# Replyzen

## Descrição

A Replyzen é uma aplicação web desenvolvida para pequenos negócios que desejam automatizar atendimentos no WhatsApp e melhorar a comunicação com clientes.

## Objetivo

O projeto foi desenvolvido utilizando ferramentas no-code e tecnologias web para demonstrar integração dinâmica, personalização visual e aplicação de padrões web fundamentais.

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* GitHub Pages
* Tally Forms
* Google Sheets
* Carrd

## Funcionalidades

* Landing page responsiva
* Formulário inteligente integrado
* Integração automática com Google Sheets
* Organização automática de leads
* Interface moderna
* Scroll suave
* Botões animados
* Responsividade para dispositivos móveis

## Integrações

O formulário Tally envia automaticamente os dados preenchidos para uma planilha Google Sheets integrada, criando um fluxo automatizado de captação de leads.

## Responsividade e Acessibilidade

O projeto foi desenvolvido pensando em dispositivos móveis e desktop, utilizando contrastes fortes, textos legíveis e organização visual para melhorar a experiência do usuário.

## Ética Digital

A aplicação coleta apenas informações essenciais para contato comercial, respeitando princípios básicos de privacidade e ética digital.
